## git init `dir`
- 初始化工作空间

## git add `file`
- 将文件添加到缓存区
> `示例`
> git add .  表示当前路径所有文件


## git commit
- 将缓存区文件添加到分支
- -m 添加提交的日志
> `示例`
> git commit -am "init" 表示先添加文件再提交文件,组合使用

## git remote add `urlname` `url`
- 添加远程路径,push的时候可使用这个`urlname`</br>
> `示例`
> git remote add origin https://github.com/pansusu/tulinglab.git
